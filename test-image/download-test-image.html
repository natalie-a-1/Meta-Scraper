Testing Steps:

  1. Download one or more images using the buttons above
  2. Save the images to your test-image folder
  3. Open VS Code with the Metadata Scraper extension installed
  4. Right-click on an image and select "View Image Metadata" to see what data is embedded
  5. Right-click again and select "Clean Image Metadata" to remove the metadata
  6. Verify the cleaning worked by using "View Image Metadata" again
